# 信息与计算科学导论实验

## 课程信息 {#detail}

这门课和 [信息与计算科学导论](iics.md) 没啥联系，主要讲的是程序设计和一些简单的算法。

## 课程评价 {#review}

有点难评。上课讲的是基础的 C++ 语法，但是课程作业是不知什么难度的 OI 题。显然是需要自学很多了。

期中期末考的时间正好和 [SICP](sicp.md) 冲突了，导致我没有见到樾哥的第一面和最后一面。
所以对这门课的印象不是很好。不过其实老师人也挺好的，就是上课怎么没人听呢，教室里跟死了一样，
~~总感觉信计这种神秘课程的课堂氛围都这样，以后也逃不掉哎~~。

考试题比较简单，考后给分也非常猛，老师被要求分数不要有区分度，也算是信计特色了。
