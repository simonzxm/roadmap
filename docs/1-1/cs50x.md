# Harvard CS50x

## 课程信息

你也许发现了这不是南京大学的课程，不过我希望他能出现在这里。

课程网站：<https://cs50.harvard.edu/x/>

课程资料和 OJ 都是公开的，哪里都能找到，可以随便学。

## 课程评价

这是一门计算机的基础课程，他涵盖了 C Python SQL HTML CSS JavaScript 等内容的基础知识，课程内容非常丰富。

这门课是我在开学前的暑假上的，当时我是有一些 Python 基础，没有接触过 C 语言。不过这个老师讲的真的非常好，
上课非常有趣，可惜我大概暑假听到 Flask 左右就听不动了。这个课的作业非常重要，建议一定要认真完成，
在我这个零基础的人眼里看来，作业难度不小，每次做都要花不少时间，但是每次做完作业都会有很大的收获。

最后有一个大作业，是自己随便做一个项目，如果想获得他的正式证书的话还需要录一段视频，这里我偷懒了（）
所以其实算是一直没学完这个课。

这门课里的知识会很容易的帮到后续的学习，他算是一门非常好的引入课程。如果实在没时间听，我建议把 Python
前面的所有内容听完，这能帮你对 C 语言、数据结构和算法有一个基础的认识。至于后面的内容，
我认为可以是用来纯粹的拓宽眼界的，不想学就算了。

至于怎么从他和 CS61A 中选，我的评价是，我们有 CS61A 的复刻了，真没必要听网课，直接去上 [SICP](sicp.md)
就行了，所以你真的暑假没事干，就看 CS50x 吧。另外，CS61A 是不学 C 语言的，所以如果你不了解 C 语言，
可能后续课程会比较吃力，或者当然你也可以选上学校的 [程序设计基础](../1-2/bp.md)。
